# Basic Linux Commands

# File and Folder Navigation Commands

 ls - This command lists the contents of a directory/folder. 
 
 pwd - pwd(print working directory), as the name suggests it prints the 
 current working directory in which user is present. 

 cd - cd(Change Directory), This command is used to change or navigate 
 through different folders. e.g - cd <Folder Name> to go back to the previous 
 folder/directory just use "cd .." command, this takes you back to previous
 folder. 

 find - find is used to find a file within multiple files in a directory. 
 e.g - find <Filename.extension> if you want to sort out files with specific 
 extension you can use (e.g - find *.txt). 

 locate - Used to locate a file and its path.  


# File and folder operations commands

 mkdir - Used to create a directory, e.g - mkdir <Directory Name> 

 rmdir - Used to Remove or delete a Directory. 

 rm - Used to delete a file. 

 cp - Used to copy the contents of one file into another. 
      e.g - cp file1 file2. 
 
 mv - Used to move or rename a file or folder. 

 touch - Used to create a single empty file. 

 file - Used to check type of the file. 

 nano - Used to open a file into text editor in terminal. 

 grep - Used to Find specific lines from a file. 

 cat - cat(Concatenate) used to print contents of a file. 

 echo - Prints any text into terminal. e.g - echo "Hello Linux" Prints "Hello 
 Linux"

 clear - clears the terminal. 


# User and Permission management Commands 

 sudo - one of the most important command, it runs any command as a super user.
 It can bypass any permission restrictions and acts as a admin user. 

 whoami - Used to check the current user. 

 passwd - Set new password for a user. 

 useradd - Adds a new user. 

 userdel - Removes a user. 

 su - Change User. 


# System monitoring and processes commands 

 kill - Terminate a process. 

 shutdown - Turn off your system. 

 man - Displays a manual for a command. 

 cal - opens Calendar. 

 htop - List and manage processes.
